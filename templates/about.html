About the DOJ Case Analysis Project

üéØ Mission

We track and analyze Department of Justice prosecutions under 18 U.S.C. ¬ß 1960 (money transmission without a license), with a particular focus on cryptocurrency-related cases.

This project creates an open dataset for researchers, journalists, and policymakers to understand how federal authorities are enforcing money transmission laws in the digital age.

üîç What We Do

Data Collection
  • Automatically scrape DOJ press releases (2009-present)
  • Filter for cases mentioning 18 USC 1960 or cryptocurrency
  • Store structured data in searchable database
  • Maintain idempotent scraping to avoid duplicates
AI Verification
  • Use Venice AI to verify case classifications
  • Identify false positives and improve accuracy
  • Extract detailed case metadata
  • Provide confidence scores for classifications

Data Ingestion Pipeline
  1. Scraping: Automated collection from DOJ press release API
  2. Filtering: Keyword-based identification of relevant cases
  3. Storage: SQLite database with structured schema
  4. Verification: AI-powered classification and validation
Classification Process
Initial Filtering
  • 18 USC 1960 mentions
  • Cryptocurrency-related terms
  • Money transmission keywords
AI Verification
  • Context analysis
  • False positive detection
  • Confidence scoring
  • Detailed classification notes

üìä Data Schema

Each case in our database contains:

Basic Information
  • title - Press release title
  • content - Full press release text
  • date - Publication date
  • url - Original DOJ URL
Classification Data
  • mentions_1960 - Boolean flag for 18 USC 1960
  • mentions_crypto - Boolean flag for cryptocurrency
  • verified_1960 - AI verification result
  • verified_crypto - AI verification result
  • classification - Detailed AI notes
